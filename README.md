# Job Search Platform

Welcome to the Job Search Platform repository! This platform is designed to help users search for job listings based on various criteria such as company name, location, experience, tech stack, and more. Users can filter job listings, view detailed job descriptions, and apply for jobs directly through the platform.

## Features

- **Search**: Users can search for job listings based on keywords, location, experience level, etc.
- **Filter**: Filter job listings based on company name, location, experience, tech stack, remote/onsite, etc.
- **Job Listings**: View detailed job listings including company name, job title, location, job description, tech stack requirements, and more.
- **Easy Apply**: Users can apply for jobs directly through the platform with just a click.
- **Infinite Scroll**: Job listings are loaded dynamically with infinite scrolling for smooth browsing.

## Technologies Used

- **Frontend**: React.js, Material-UI
- **State Management**: Redux
- **API Integration**: Axios
- **Styling**: CSS, MUI-components
